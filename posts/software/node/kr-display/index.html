A Metor based application for displaying production data from an SQL server.

This project is similar to the Node-SCADA-System however it doesn’t collect the data from the PLCs directly. The production trial of this system used KEPserver to collect data from Mitsubishi and Toyopuc PLCS and this project to display the data at the equipment.
